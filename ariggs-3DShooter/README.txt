In this folder is my submission for the Unity Surivival Shooter project. An explaination of the content is below.

	Survival_Shooter
This is the Unity Project folder. It contains the original completed tutorial as well as my version (Level 1).

	Changes
A text file that keeps track of the changes I made to the original tutorial, after its completion. 

Controls for the game:
	WASD to move
	Mouse to rotate/Click to shoot
	Escape to exit
	L to restart at any time
